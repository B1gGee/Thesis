# Stock Thesis Monitor 📈

**One-click automated stock thesis dashboard with custom investment thesis.**

Built exactly to your specification:
- Top summary + autonomous recommendation (Strong Buy / Hold / Monitor / Sell) with auto-color
- Entry-Level (5-10 min scan), Mid-Level (20-30 min), High-Level (deep dive)
- Columns: **Metric/Query** | **Value/Notes** | **Flag** (Green/Orange/Red) | **Source/Date**
- Triggers with color-coded bull/bear signals
- Conviction Score (+1 green, 0 orange, -1 red)
- "Why I Own It" / "What Must Stay True" / "What Would Change My Mind" bonus block
- Fully editable custom thesis
- Live data from Yahoo Finance (cross-referenced)
